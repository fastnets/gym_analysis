# Анализ оттока клиентов фитнес-клуба

## Описание проекта

Данный проект посвящён анализу данных о клиентах фитнес-клуба с целью выявления факторов, влияющих на отток, и построению модели для прогнозирования вероятности ухода клиента. В работе применяются методы исследовательского анализа данных (EDA), feature engineering и различные алгоритмы машинного обучения.

## Используемые технологии

- pandas, numpy — обработка данных
- matplotlib, seaborn — визуализация
- scikit-learn — машинное обучение и препроцессинг
- xgboost — градиентный бустинг
- joblib — сохранение моделей

## Описание этапов работы

### 1. EDA и исследовательский анализ

- Визуализация распределения целевой переменной (отток).
- Анализ корреляций между признаками и оттоком.
- Изучение распределений ключевых признаков (возраст, длительность и частота тренировок, тип тренировки и др.).
- Визуализация различий между группами клиентов.

### 2. Feature Engineering

- Кодирование категориальных признаков.
- Добавление новых признаков: BMI, интенсивность тренировок (Calories_per_hour), возрастные группы.
- Оценка корреляции новых признаков с оттоком.

### 3. Построение и сравнение моделей

- Обучение моделей разных семейств: логистическая регрессия, Random Forest, XGBoost, нейронная сеть.
- Оценка качества моделей с помощью 5-кратной кросс-валидации.
- Визуализация важности признаков и сравнения метрик моделей.

### 4. Применение модели

- Сохранение обученной модели.
- Пример предсказания для новых клиентов (группа риска/не в группе риска).

## Краткие выводы

- Ключевые признаки риска оттока: частота посещений, длительность тренировок, уровень опыта, количество сожжённых калорий.
- Ансамблевые методы (Random Forest, XGBoost) показывают наилучшие результаты по точности и стабильности.
- Модель может быть использована для раннего выявления клиентов с высоким риском ухода и разработки персональных программ удержания.

## Как запустить проект

1. Клонируйте репозиторий:

```git clone https://github.com/fastnets/gym_analysis```

2. Установите зависимости:

```pip install -r requirements.txt```

3. Откройте Jupyter Notebook:

```jupyter notebook gym_churn_analysis.ipynb```

4. Запустите все ячейки по порядку для воспроизведения анализа.
